#Lexicon Detector

An advanced, interactive language detection tool that identifies 50+ global languages with high accuracy and an elegant UI.



#Overview

Lexicon Detector is a browser-based language identification tool built using HTML, CSS, and JavaScript.
It allows users to paste or type text, and the system intelligently predicts the language along with the confidence level and top-matching alternatives.

This project also features a clean UI, sample text buttons, a confidence bar, and detection history.

#Features

Accurate Language Detection

-> Detects over 50 languages

-> Confidence score visualization

-> Top 3 language match results with percentage

-> Simple detection logic (extendable to ML APIs)

 Fast & Interactive UI

-> Real-time character & word count

-> Sample text shortcuts (French, Spanish, English, German)

-> Clean, modern, responsive UI design (CSS-based)

 Detailed Results Panel

-> Detected language

-> Confidence meter

-> Top matches with confidence

-> Detection history(last 5 queries

Tech Stack

-> HTML – UI layout

-> CSS – Styling & responsive grid design

-> JavaScript – Detection logic, UI updates, history management
